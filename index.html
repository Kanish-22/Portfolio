My Projects

Patient Blood Oxygen Level Monitoring System using IOT

This projects uses sensors that allow to detect blood oxygen level of a person sensing even if the person is at home. The sensor is then interfaced to a microcontroller that allows checking SpO2 readings and transmitting them over internet. The user may set the high as well as low levels of blood oxygen saturation limit.As soon as patient SpO2 level goes above a certain limit, the system sends an alert to the controller which then transmits this over the internet and alerts the doctor as well as concerned users.The system alerts the concerned person through call, Email,telegram and message. Thus concerned ones may monitor heart rate as well get an alert about the patient immediately from anywhere and the person can be saved on time.

Netflix Clone using HTML and CSS



I developed a Netflix Clone using HTML and CSS to replicate the popular streaming platform's user interface. The project aimed to recreate Netflix's familiar browsing experience, complete with responsive design for seamless usage across devices. Key features include a navigable layout with categories and genres, thumbnail displays for movies and TV shows with hover effects for additional information, and a sign-in page for user authentication. Implementing pixel-perfect design and ensuring responsiveness challenged my attention to detail and CSS skills. This project has been invaluable in honing my front-end development skills and understanding the complexities of creating immersive user experiences in web development. The initial design and layout planning were done using Figma.
